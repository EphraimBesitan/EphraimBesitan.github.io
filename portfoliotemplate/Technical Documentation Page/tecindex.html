Git Documentation

Introduction to Git

Git is a distributed version control system used for tracking changes in source code during software development.

Basic Git Commands

  • git init - Initialize a new Git repository
  • git add <file> - Add file(s) to staging area
  • git commit -m "message" - Commit changes to the repository
  • git status - Check the status of working directory adn staging area
  • git log - View commit history

Branching in Git

Branching allows you to diverge form the main line of development and continue working without altering the main codebase.

Introduction to HTML5

HTML5 is the latest version of Hypertext Markup Language, the code that describes web pages. It's the fifth version of the HTML standard, and it comes with many new features to improve the structure of web documents and enhance the user experience.

CSS (Cascading Style Sheets)

CSS is a style sheet language used for describing the presentation of a document written in HTML. It allows web designers to control the layout, colors, fonts, and other visual aspects of a web page.

JavaScript

JavaScript is a high-level programming language that is commonly used alongside HTML and CSS to create interactive and dynamic web content. It enables developers to add behavior to web pages, manipulate the DOM (Document Object Model), handle events, and much more.

Linux for Windows

Linux for Windows, also known as Windows Subsystem for Linux (WSL), allows users to run a Linux distribution directly on Windows without the need for a virtual machine. WSL provides a full-fledged Linux environment that enables developers to run native Linux command-line tools and applications.

Introduction to Flexbox

Flexbox, short for Flexible Box Layout, is a CSS layout model that provides a more efficient way to design and align elements within a container. It offers a set of properties for distributing space among items in a container and aligning them in various ways, making it easier to create responsive and flexible layouts.
